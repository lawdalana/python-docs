# Python Email

Poll for mails and return the Number of unread mails in your mail box.

Setting Environment variables required.

```sh
$ export GMAIL_USER=yourmailid@gmail.com
$ export GMAIL_PASS=yourGmailPassword
```

Running the Script

```sh
$ ptyhon checkmail.py
```
